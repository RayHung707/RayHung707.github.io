Intelligent Wearable Mobile Assistive Device

Semantic Segmentation · DeepLabv3+
Summary
A wearable assistive system that helps visually impaired users walk outdoors, combining an obstacle-avoidance guide cane with a Jetson Nano vision system. The vision system runs two deep-learning modules — sidewalk detection and store-sign recognition — and gives guidance through easy-to-understand voice prompts.
Tools
DeepLabv3+ (Xception-65 backbone) for sidewalk segmentation; YOLOv4 for store-sign recognition; NVIDIA Jetson Nano; Arduino with an ultrasonic sensor; text-to-speech voice output.
My Role
Built the DeepLabv3+ sidewalk-detection module — prepared the data, trained the segmentation model, deployed it on the Jetson Nano, and turned the results into voice guidance that keeps the user on the sidewalk.
Results
Reached a working real-time prototype on the Jetson Nano. The results were promising as a proof of concept, though accuracy still dropped in harder real-world conditions, leaving clear room to improve.
Impact
Reinforced my interest in applying computer vision to safety and accessibility, and taught me that running a model reliably in real time on constrained hardware is as much engineering as the model itself.
DeepLabv3+ segmenting the walkable sidewalk area, running on the Jetson Nano.

Automatic Meter Reading System

Object Detection · YOLOv4
Summary
A YOLOv4-based system that reads mechanical utility meters from a photo — a low-cost bridge between manual reading and a full smart-meter rollout. We built and compared two approaches: a two-stage pipeline (YOLOv4 finds the digit row, then Tesseract OCR reads it) and an end-to-end model (YOLOv4 detects and classifies every digit in one pass).
Tools
YOLOv4, Tesseract OCR, Python; deployable on an NVIDIA Jetson Nano for edge use.
My Role
Built the end-to-end YOLOv4 pipeline. Helped collect and label a dataset of water, electric, and gas meters across varied lighting and angles, then trained the model on ten digit classes (0–9) and tested the messy real-world cases.
Results
Read all eight digits correctly across different meter brands in clear photos at moderate angles. Confidence dropped sharply (0.40–0.56) under heavy dirt or rotation past ~45°, which pointed to data quality as the main driver of model performance.
Impact
Model quality came from being careful with the data, not from tuning the model — and a model has to be judged on real conditions, not a clean test set.
End-to-end YOLOv4 reading a residential water meter, each digit boxed with its confidence score.
